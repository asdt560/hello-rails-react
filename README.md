### Prerequisites

In order to run this project you need: ``` Ruby ```, ``` Rails ``` and a Database such as PostgreSQL

### Setup

Clone this repository to your desired folder.

### Install

Install this project with:

```
  bundle install
  yarn install
  rails db:migrate
  rails db:seed
```
